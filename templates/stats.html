Correlation Map

The HeatMap below shows how all the feartures are correlated to each other

We can clearly infer from the HeatMap that Mileage, production year and transmission type have a highest positive correlation with the target variable i.e. Price of the vehicle which means that if the value of these any of these three feature increases the price of the vehicle will also increase.

We can also infer from the HeatMap that color and engine Capacity have a negetive correlation with the target variable i.e. Price of the vehicle which means that if the value of these any of these features increases the price of the vehicle will decrease.
